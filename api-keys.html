Resume Tailor — API keys

Short guide to creating keys for each provider. Your keys stay in your browser when you use the app.

Privacy: Your API key never leaves your browser in a way Resume Tailor can see. This project does not run a server that receives, logs, or stores your key. Requests go from your browser directly to the provider you choose.
Claude (Anthropic) 🤖

Best for polished, nuanced writing and structured outputs.

~$0.01–0.02 per full generationNo free tier
  1. Open Anthropic Console and sign in.
  2. Go to API Keys.
  3. Create a key and copy it.
  4. Paste into Resume Tailor Step 1.

ChatGPT (OpenAI) 💬

Widely familiar; strong general-purpose model access.

~$0.01–0.02 per full generationNo free tier
  1. Open OpenAI Platform and sign in.
  2. Go to API keys.
  3. Create new secret key and copy it.
  4. Paste into Resume Tailor Step 1.
Open API Keys →
Gemini (Google) ✨

Budget-friendly; good for high-volume or experimentation.

~$0.001 per generation (varies)Free tier available
  1. Open Google AI Studio.
  2. Use Get API key (or API keys).
  3. Create a key in the Google Cloud / AI Studio flow.
  4. Paste into Resume Tailor Step 1.

Ollama (Local) 🦙

Run open models on your machine — no cloud API key for inference.

$0 forever (after setup)Free — no key
  1. Download Ollama for your OS.
  2. Install and pull a model (e.g. mistral-nemo).
  3. Run Ollama locally; use the local HTML build with Ollama enabled.
  4. No API key — select Ollama in Step 1.
